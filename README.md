# Taximeter

A Kotlin Multiplatform (Android + iOS) taximeter app using Compose Multiplatform.
Small testproject to test the capabilities of Claude Code

### Features

- **Cross-platform**: Runs on both Android and iOS devices
- **GPS Tracking**: Real-time location tracking for accurate distance calculation
- **Trip Calculation**: Automatic fare calculation based on distance and time
- **Passenger Management**: Add and manage multiple passengers
- **Route History**: Store and view trip history
- **Local Data Storage**: All data stored locally on device
- **Privacy-focused**: No user data collection or external data transmission
- **Modern UI**: Clean, responsive interface using Compose Multiplatform
- **Offline Support**: Works without internet connection
